# Padrões de Micro Serviços

## Single Service Database ou Database Per Service

- Separa os bancos de dados de micro serviços de forma que cada serviço possua um banco

## Shared Service Database

- Quando um banco de dados precisa ser obrigatoriamente compartilhado, mas devem ser acessados de forma independentes.

Sempre buscar o Single Service Database
